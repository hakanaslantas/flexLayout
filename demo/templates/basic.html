usage:    $('some-element').flexlayout([layoutConfig], {options});
[layoutConfig]
Describe your layout configuration by using an array, in which every element describes the dimension of a block in the layout.
var layoutConfig = ['1, 3, 5'];
The dimension ratio among three blocks is 1:3:5; i.e. the height/width of blocks has a ratio of 1:3:5.

var layoutConfig = ['50px', '2', '5', '3em'];
The first and the last block has a fixed length/width.
      The dimension of the two middle blocks has a ratio of 2:5(after deducting the total length/width with 50px and 3em).

Give attributes to the newly generated blocks, simply by separate attributes and dimension with ":".
var layoutConfig = ['1:#left .left', '4:class="sc" id="sn" data-attr="some"', '5', '3em:.right1 .right2'];
Either give the attribute with inline style or give it with jQuery selector style.

{options}
options: 
			
				{
					/*defines the direction of split(horizontally or vertically); default: 'h'*/
					dir: 'h' or 'v', 
					/*defines whether the length/width of the blocks can be adjusted by dragging bars; default: false*/
					adjust: 'true' or 'false' 
					/*defines the style of divide bars between created blocks 
					 *takes a css object, a string represents class name or false(no bars)
					 *default: {flex: '0 0 2px', 'background-color': '#ddd'}*/
					bars: {css object}, 'string of class name' or false
					/*defines the height of the requesting element; default: '100%';*/
					height: 'string',
					/*defines the width of the requesting element; default: '100%'*/
					width: 'string',
				}
Note:
  • When you want to give height/width to the divide bars, give it as flex: '0 0 height/width'.
    Flexbox does not work well with height and width property. For more check More on flexbox(TBD).
  • If you give a fixed height/width to a block, that block cannot be adjusted even you have specified adjust is true.
