flexLayout v0.2.1
A light weight and highly customizable jQuery layout/split plugin.
Plugin Features
Easy to use -- simple configurations.
Highly customizable -- easy configuration for every nested layer and use your own attributes and styles right in the code.
Resizable -- individually resizable and whole block resizable.
Easy to use with Backbone -- just assign el with generated layouts.
Work on all major browsers -- implemented with flexboxes.
Lightweight -- 1kb when gzipped.
